# 20201211 记录

### 域名等级

以博客地址为例：`https://ctts.github.io/`

**一级域名**

域名有多个等级，必备的是一级域名，常见的如` .com` ` .cn` ` .io` 等等

对应例子里的 `.io`

**二级域名**

所谓的二级域名实际上就是一个一级域名下面的主机名。

对应`.github.io`

**子域名**

也就是二级域名之前的域名。
