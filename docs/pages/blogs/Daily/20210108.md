# 20210108 记录

### cookie 默认保存时间有三种类型。

- -1 是 session（关闭浏览器自动清除）
- 0 是 立刻删除
- \> 1 是设置有效时间

### 移动端正则中存在 <= 可能导致部署失败和移动端白屏

有问题的写法：
```javascript
/(?!$)(?<=^(\d{4})+)/g
```
依然有问题的写法：
```javascript
var reg =new RegExp("(\\B)(?<=\\b(\\d{4})+)","g");
```
正确的写法：
```javascript
.replace(/\s/g, '').replace(/(.{4})/g, "$1 ")
```
